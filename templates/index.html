Get the API to check your available attempts for various exams.


Exams covered right now

National Defence Academy
Combined Defence Services Examination
Air Force Common Admission Test
Civil services examination


The API can be used from the URL itself which returns a JSON response


The Pattern for the URL


Lists for the variables in the URL :-

https://upscapi.pythonanywhere.com/{exam}/{gender}/{preference}/{dob}

where exam, gender, preference and dob are variables to be entered by the user

exam= ['nda', 'cds', 'afcat', 'cds']
gender=['male', 'female']
if exam is NDA preference=['ima', 'ina', 'afa']
if exam is CDS preference= ['ota','ina','afa', 'ima']
if exam is afcat preference= ['afa_gd', 'afa_flying']
Note:- DOB should always be in YYYY-MM-DD form eg. 2002-10-28

example response:-
https://upscapi.pythonanywhere.com/nda/male/afa/2002-10-28
